# Spectrophotometric standard stars

Reference data for identifying and flux-calibrating standard-star exposures. LLAMAS headers do
not flag standards — they carry `OBS TYPE = SCIENCE` like any target — so they are found by
crossmatching the pointing against the coordinates here.

Generated by `python -m llamas_pyjamas.Scripts.fetch_standards`; re-run to refresh.

## Contents

- **`index.dat`** — one row per standard: `name`, `ra_deg`, `dec_deg` (decimal J2000), `vmag`,
  `sptype`, `flux_scale`, `flux_file`. 68 stars from the ESO
  [RA-ordered list of spectrophotometric standards](https://www.eso.org/sci/observing/tools/standards/spectra/stanlis.html).
  `flux_scale` multiplies column 2 of the flux file to give erg/s/cm²/Å.
- **`flux/xsh_f<name>.dat`** — XShooter reference spectra (Moehler+2014), preferred where
  available. Two columns: wavelength (Å), flux (erg/s/cm²/Å; `flux_scale` = 1). Binned to ~1 Å
  from the 0.1 Å originals. **3000–25000 Å** coverage. See `flux/aaareadme.Xshooterspec`.
- **`flux/f<name>.dat`** — Oke (1990) reference spectra (STScI CALOBS via ESO/STECF), used when
  there is no XShooter spectrum. Four columns: wavelength (Å), flux (10⁻¹⁶ erg cm⁻² s⁻¹ Å⁻¹;
  `flux_scale` = 1e-16), flux (mJy), bin (Å). **3200–9200 Å**. See `flux/aaareadme.oke`.

## Coverage

Coordinates cover all 68 ESO optical standards; reference spectra are bundled for 30 (6 from
XShooter, 24 from Oke). The may26 standards: **Feige110** uses XShooter (→25000 Å, so the red
sensfunc reaches the data limit ~10000 Å), while **GD108** is Oke-only (→9200 Å — it is not in
the XShooter set, so calibration past 9200 Å would need a CALSPEC spectrum or extrapolation).
Prefer a XShooter standard (Feige110, GD71, GD153, EG274, LTT3218, LTT7987) when red coverage
matters. The remaining stars have coordinates but no bundled spectrum (their tables live in
ESO's `ctiostan/`, `hststan/`, `wdstan/` archives, with inconsistent naming); extend
`fetch_standards.py` if a future night needs one.

## Consumed by

`llamas_pyjamas.Flux.fluxStandards` loads `index.dat` and crossmatches pointings (default 30″).
