llamas_pyjamas.GUI.guiExtract.GUI_extract(file, flatfiles=None, bias=None)[source]

Extracts data from a FITS file using calibration files and saves the extracted data. Parameters: file (str): Path to the FITS file to be processed. Must have a .fits extension. flatfiles (str, optional): Path to the flat files for generating new traces. Defaults to None. biasfiles (str, optional): Path to the bias files for calibration. Defaults to None.

Returns: None
