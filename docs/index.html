Welcome to LLAMAS Pyjamas documentation!

LLAMAS Pyjamas is a comprehensive Python package for processing and analyzing data from the LLAMAS (Large Lens Array Multi-Object Spectrograph) instrument. This package provides a complete pipeline for spectroscopic data reduction, including tracing, extraction, wavelength calibration, and analysis tools.

Features

  • Complete Data Reduction Pipeline: From raw CCD images to science-ready spectra

  • Multi-Channel Support: Handles blue, green, and red spectrograph channels

  • Optimal Extraction: Advanced fiber extraction algorithms for maximum signal-to-noise

  • Wavelength Calibration: Thorium-Argon arc line identification and wavelength solutions

  • White Light Imaging: Generate white light images and fiber maps from extracted spectra

  • Quality Assurance: Comprehensive QA tools and visualizations

  • Parallel Processing: Ray-based parallel processing for improved performance

Quick Start

Installation

# Clone the repository
git clone https://github.com/your-org/llamas-pyjamas.git
cd llamas-pyjamas

# Install dependencies
pip install -e .

Basic Usage

from llamas_pyjamas.Trace import traceLlamas
from llamas_pyjamas.Extract import extractLlamas
from llamas_pyjamas.Arc import arcLlamas

# Trace fiber positions
trace = traceLlamas.TraceLlamas()
trace.traceSingleCamera(data_object)

# Extract spectra
extractor = extractLlamas.ExtractLlamas(trace, hdu_data, header)

# Wavelength calibration
arcLlamas.arcSolve(arc_extraction_file)

Package Overview

The LLAMAS Pyjamas package is organized into several modules:

Core Processing Modules

  • llamas_pyjamas.Trace - Fiber tracing and profile fitting

  • llamas_pyjamas.Extract - Spectral extraction algorithms

  • llamas_pyjamas.Arc - Wavelength calibration and arc line identification

  • llamas_pyjamas.Flat - Flat field processing and correction

  • llamas_pyjamas.Bias - Bias subtraction and dark current correction

Data Handling and I/O

  • llamas_pyjamas.File - FITS file I/O and data handling

  • llamas_pyjamas.Image - White light image generation and processing

Analysis and Visualization

  • llamas_pyjamas.QA - Quality assurance and diagnostic tools

  • llamas_pyjamas.Flux - Flux calibration and throughput analysis

  • llamas_pyjamas.Utils - Utility functions and helpers

Post-Processing

  • llamas_pyjamas.Postprocessing - Advanced data products

User Interface

  • llamas_pyjamas.GUI - Graphical user interface components

About LLAMAS

The Large Lens Array Multi-Object Spectrograph (LLAMAS) is an innovative instrument designed for efficient multi-object spectroscopy. This Python package provides the complete data reduction pipeline for LLAMAS observations, enabling astronomers to extract high-quality scientific spectra from raw CCD images.

For more information about the LLAMAS instrument and its scientific applications, please refer to the instrument documentation and published papers.
